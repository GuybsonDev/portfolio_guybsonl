API Crescimento do PIB

Este projeto visa fornecer informações detalhadas sobre o crescimento do PIB dos países africanos ao longo dos anos, utilizando dados atualizados e uma API robusta para consultas eficientes.

Visão Geral

Desenvolvi uma API funcional com parâmetros e filtros avançados para aprimorar meus conhecimentos em desenvolvimento de APIs. A API utiliza uma base de dados sobre o crescimento do PIB dos países africanos, abrangendo 63 anos de histórico (1960-2023), extraída do Kaggle.

Tecnologias Utilizadas

  • Linguagem: Python
  • Framework: Flask
  • Manipulação de Dados: Pandas
  • Ambiente de Desenvolvimento: Replit
  • Testes de Requisição: requests

Funcionamento

A API permite dois modos de consumo:

  • Consulta completa: Retorna todos os dados disponíveis.
  • Consulta filtrada: Permite buscar informações com base em um período específico (ano), utilizando parâmetros personalizados.

Desenvolvimento e Testes

Para manipulação e análise eficiente da planilha, utilizei a biblioteca Pandas. A implementação da API com Flask garantiu uma solução ágil e simples. O projeto foi desenvolvido no ambiente Replit, facilitando o acesso remoto à API.

Os testes foram realizados com a biblioteca requests, garantindo que as requisições HTTP fossem processadas corretamente e validando o consumo dos dados de maneira eficiente.
